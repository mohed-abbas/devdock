## Interaction Contracts

### Preview Port Field (CreateEnvironmentDialog)

- Field type: `<Input type="number" />` with shadcn Label
- Label: "Preview Port"
- Placeholder: "3000"
- Helper text: "Port your app listens on inside the container (optional)"
- Validation: integer 1–65535 or empty. Empty = no preview URL.
- Placement: below Branch selector, before submit button.

### Auto-Scroll Toggle (Logs Page)

- Toggle Button. Label: "Auto-scroll" with icon `ArrowDownToLine`.
- Default state: ON (active/pressed appearance via `variant="secondary"`).
- When ON: new log lines scroll the view to bottom automatically.
- When OFF: view stays at current scroll position. Label changes to "Scroll paused" visually via icon state.

### Clear Button (Logs Page)

- Button `variant="ghost" size="sm"`. Icon: `Trash2`. Label: "Clear".
- Clears the in-memory log buffer for the current view. Does not clear Docker logs.
- No confirmation dialog — non-destructive (logs are re-streamed on reconnect).

### Log Streaming Connection States

| State | Indicator |
|-------|-----------|
| Connecting | Spinner in header area, log area blank |
| Connected | No indicator (streaming begins) |
| Reconnecting | Amber overlay text "Reconnecting..." centered in log area |
| Disconnected | Error overlay with "Retry" button (matches terminal ReconnectOverlay pattern) |
