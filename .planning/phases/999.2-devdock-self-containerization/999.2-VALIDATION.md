Test exports `DATABASE_URL` only, which the wait loop ignores. | Patch entrypoint to honor `POSTGRES_PORT` (already used implicitly by `pg_isready -p`); patch test to export `POSTGRES_HOST=127.0.0.1` and `POSTGRES_PORT=55432`. |
| GAP-2 | 03-T1 / 10-T1 | `docker compose up --build` fails at `RUN npm run build` with `Invalid environment variables: { DATABASE_URL: [ 'Required' ], AUTH_SECRET: [ 'Required' ] }` | Next.js 15 page-data collection at build time evaluates `src/lib/config.ts` which calls `loadConfig()` → throws on missing vars. Real values come from compose `env_file` at runtime. |
