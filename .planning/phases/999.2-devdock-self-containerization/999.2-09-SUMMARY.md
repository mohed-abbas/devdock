## HUMAN-UAT Self-Containment Assessment

The runbook is self-contained — no external dependencies required to follow it:

- All shell commands are standard Linux utilities (`sudo`, `docker`, `nginx`, `ss`, `openssl`)
- All paths either use the repo checkout location (`/home/murx-dev/devdock`) or are clearly marked "adjust to your VPS"
- Admin password hash generation uses `npm run hash-password` from the repo itself
- Rollback section uses only `docker compose down` + `sudo systemctl start devdock` (both already present on the VPS)
