<objective>
Fix the dev-time 500 caused by Turbopack failing to resolve `@import "shadcn/tailwind.css";` in `src/app/globals.css`.

Purpose: The shadcn package's `package.json` only exposes `./tailwind.css` through the `style` export condition, which Turbopack's CSS resolver does not honor when handling a CSS `@import`. We map the bare specifier to the real on-disk path so the import works without modifying upstream code or the source CSS.

Output: An updated `next.config.ts` containing a `turbopack.resolveAlias` entry. The dev server boots, the homepage returns 200, and the login page renders with the word "Username" present.
</objective>
